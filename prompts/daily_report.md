# 股票追踪简报 - 增量更新模板

请为我生成一份 {date} 的股票追踪增量简报。

## 追踪股票
{stocks}

## ⚠️ 重要：这是增量报告

**你已经阅读了上次的报告**，今天的任务是：
- ✅ 只写**相对于上次报告的变化**
- ✅ 突出**新发生的事件**和**新的趋势**
- ❌ 不重复上次已经说过的基础信息
- ❌ 不重复公司介绍、行业背景等静态内容

## 报告结构

### 1. 今日表现对比（必须用表格）

| 股票代码 | 股票名称 | 今日价格 | 涨跌幅 | 成交量变化 | 备注 |
|---------|---------|---------|-------|-----------|------|
| XXX | XXX | ¥XX.XX | +/-X.X% | ↑/↓ XX% | 重点关注/平稳 |

**说明**：
- 涨跌幅：今日相对昨日收盘
- 成交量变化：相对近期平均水平
- 备注：标注需要重点关注的股票

### 2. 重点变化分析

**仅分析以下类型的变化**：

#### A. 显著价格变动（涨跌幅 > 3%）
```
{股票名称}({代码})：{涨/跌} X.X%
- 变动原因：{具体原因}
- 市场解读：{分析}
- 后续关注：{观察点}
```

#### B. 成交量异常（相比5日均量 > 50%）
```
{股票名称}：成交量放大/萎缩 X%
- 资金动向：{主力/散户}资金{流入/流出}
- 可能原因：{分析}
```

#### C. 技术面突破/跌破
```
{股票名称}：{突破/跌破}{关键位}
- 技术意义：{解读}
- 操作建议：{关注/观望/等待}
```

**如果所有股票都没有显著变化**：
```
今日追踪股票整体表现{平稳/震荡}，涨跌幅均在±2%以内，
成交量维持常态，无需特别关注。
```

### 3. 今日新增信息

**只列出今天新发生/新发布的内容**：

- **公司公告**：{公司} 发布 {公告类型}，主要内容...
- **行业新闻**：{事件描述}，可能影响...
- **政策动态**：{政策内容}，对板块的影响...

**如果没有新信息**：
```
今日无重要公司公告或行业新闻。
```

### 4. 趋势变化提示

**仅在出现趋势转变时填写**：

- **{股票名称}**：{原趋势} → {新趋势}
  - 转变信号：{指标/事件}
  - 建议关注：{后续观察点}

**如果趋势延续**：
```
各股票延续{上次报告中的趋势}，未出现明显转折信号。
```

### 5. 明日关注

- 📅 **事件日历**：{即将发生的事件}
- 📊 **技术关键位**：{股票}关注{价格位}的{支撑/压力}
- ⚠️ **风险提示**：{新出现的风险点}

## 输出要求

### 字数与内容
- 总字数：**300-500字**
- 如果变化很少：200字即可
- 如果变化显著：可适当增加到500字

### 增量原则（关键！）

✅ **应该写的**：
- 今日新数据（价格、成交量）
- 相对昨日的变化
- 新发生的事件
- 新出现的趋势信号
- 新的风险点

❌ **不应该写的**：
- "公司是做什么的"（上次报告已说明）
- "行业背景介绍"（上次报告已说明）
- "长期投资逻辑"（上次报告已说明）
- 重复上次报告的内容

### 格式要求
- 使用 Markdown 格式
- 表格必须规范对齐
- 重要数据使用**加粗**
- 使用 emoji 提升可读性
- 段落简洁，避免冗长

## 示例：好的增量报告

```markdown
# 2026-01-23 股票追踪简报

## 今日表现

| 股票代码 | 股票名称 | 今日价格 | 涨跌幅 | 成交量 | 备注 |
|---------|---------|---------|-------|-------|------|
| 688388 | 嘉元科技 | ¥45.80 | **+5.2%** | ↑80% | 🔥重点 |
| 601318 | 中国平安 | ¥51.20 | -0.3% | 持平 | 平稳 |

## 重点变化

**嘉元科技放量上涨**
- 今日放量突破¥45关键阻力位，成交量较昨日放大80%
- 原因：公司中标某新能源项目订单（今日公告）
- 后续：关注能否站稳¥45，下一目标¥48

**其他股票**：中国平安等保持震荡，无显著变化。

## 今日新闻

- 嘉元科技：公告中标3亿元新能源项目

## 明日关注

- 嘉元科技能否站稳¥45
```

## 反面示例：不要这样写

❌ **错误1：重复基础信息**
```
嘉元科技是一家从事锂电池铜箔生产的企业，主要产品包括... 
（这些上次报告已经说过了！）
```

❌ **错误2：没有对比**
```
今日股价¥45.80
（没说相对昨日的变化！）
```

❌ **错误3：冗长啰嗦**
```
今日嘉元科技股价表现强劲，在市场整体震荡的背景下，
公司凭借优异的基本面和行业地位... （300字）
（太啰嗦了！）
```

✅ **正确写法**
```
嘉元科技今日+5.2%，放量突破¥45，中标3亿订单。
```
