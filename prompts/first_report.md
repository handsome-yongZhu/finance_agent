## 调研要求

使用 MCP 工具搜索这些股票的详细信息，包括：

1. **公司基本信息**
   - 公司全称、主营业务
   - 所属行业、行业地位
   - 核心竞争力

2. **关键指标**
   - 当前股价、市值
   - 市盈率 (PE)、市净率 (PB)
   - 其他估值指标

3. **市场表现**
   - 近期股价走势（1周、1月、3月、1年）
   - 成交量变化趋势
   - 技术面分析

4. **财务与公告**
   - 最新财报关键数据
   - 近期重要公司公告
   - 业绩预告或变动

5. **投资分析**
   - 投资亮点和机会
   - 潜在风险和挑战
   - 行业趋势影响

## 输出格式

生成一份完整详细的专业报告，使用 Markdown 格式。
报告应结构清晰，数据准确，分析客观。
